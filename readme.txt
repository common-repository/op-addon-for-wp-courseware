This plugin adds integration between OptimizePress and WP Courseware to allow customization in course units.

== Description ==
[Fly Plugins](https://flyplugins.com) presents [WP Courseware](https://flyplugins.com/wp-courseware) [OptimizePress](https://www.optimizepress.com/).

= Do you want to use the power of OptimzePress and create custom course units? =
The OptimzePress integration for WP Courseware is designed to give you control of creating custom layouts for your course units.

= OptimziePress Addon for WP Courseware Plugin =
[youtube https://www.youtube.com/watch?v=lfTzXLpfSTk]

= Shortcode =
Simply apply the [op-wpcourseware] shortcode to output the WP Courseware navigation box.

= Disclaimer =
This is not an actual theme. It is only the integration, or extension to allow you to customize WP Courseware units with OptimzePress.

== Installation ==

1. Upload the `OptimizePress Addon for WP Courseware` folder into the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
